# Estruturas de controlo condicional

## if

```python
if condition:
    sstatement(s)
elif condition:
    statement(s)
...
else:
    statement(s)
```

## Conditional expressions (ternary)

```python
var = value if condition else other_value
```

## Chained operations (Assigment and conditions)

```python
var = value == other_value == other_other_value
var = value <= other_value <= other_other_value
```
